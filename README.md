# Linux 面板与工具安装脚本

这是一个功能强大的Linux服务器管理脚本，可以一键安装常用的面板和工具。

## 功能特点

1. **宝塔面板安装** - 支持CentOS、Ubuntu、Debian系统
2. **哪吒监控面板** - 服务器监控和性能分析
3. **Docker安装** - 包含Docker常用工具
4. **服务器信息** - 显示详细的系统信息
5. **Docker功能菜单** - 包含多种常用服务的一键安装

## 安装的服务

### Docker功能菜单包含：
- **ELK套件** (Elasticsearch + Logstash + Kibana) - 日志分析平台
- **MySQL** - 关系型数据库
- **Nginx** - Web服务器
- **Redis** - 内存数据库
- **WordPress** - 博客平台

## 使用方法

### 1. 下载脚本
```bash
wget https://raw.githubusercontent.com/your-repo/linux_panel_installer.sh
```

### 2. 赋予执行权限
```bash
chmod +x linux_panel_installer.sh
```

### 3. 以root权限运行
```bash
sudo bash linux_panel_installer.sh
```

### 4. 或者直接运行
```bash
sudo ./linux_panel_installer.sh
```

## 系统要求

- **操作系统**: CentOS 7+, Ubuntu 18.04+, Debian 9+
- **内存**: 至少1GB (推荐2GB+)
- **磁盘空间**: 至少10GB可用空间
- **网络**: 需要连接互联网以下载安装包

## 功能详细说明

### 1. 宝塔面板
- 自动识别系统类型并选择正确的安装脚本
- 安装完成后显示面板访问地址
- 提供默认密码获取方法

### 2. 哪吒监控面板
- 需要提前准备好面板地址和通信密钥
- 自动下载并运行官方安装脚本
- 提供详细的配置说明

### 3. Docker安装
- 支持CentOS/Ubuntu/Debian系统
- 自动配置Docker仓库
- 安装后测试Docker运行状态
- 包含丰富的Docker管理功能

### 4. Docker功能菜单
- **ELK安装**: 配置完整的日志分析环境
- **MySQL安装**: 支持自定义密码和数据目录
- **Nginx安装**: 包含配置文件管理和网站目录
- **Redis安装**: 支持密码保护和数据持久化
- **WordPress安装**: 包含MySQL数据库的完整WordPress环境

### 5. Docker管理功能
- 查看所有容器状态
- 启动/停止/重启/删除容器
- 查看容器日志
- 进入容器终端
- 监控Docker资源使用情况

## 注意事项

1. **备份重要数据**: 在安装前请备份重要数据
2. **网络连接**: 安装过程需要稳定的网络连接
3. **系统资源**: 某些服务（如ELK）需要较多系统资源
4. **安全设置**: 安装完成后请修改默认密码
5. **防火墙**: 确保相关端口已开放

## 常见问题

### Q1: 脚本无法运行怎么办？
A: 确保以root权限运行，并检查系统是否满足要求。

### Q2: 安装过程中断怎么办？
A: 重新运行脚本，大部分安装过程会自动继续。

### Q3: Docker容器无法启动怎么办？
A: 检查端口是否被占用，磁盘空间是否充足。

### Q4: 如何卸载已安装的服务？
A: 使用Docker命令或相应的卸载脚本。

### Q5: 如何查看服务的运行状态？
A: 使用 `docker ps` 查看容器状态，或使用系统服务管理命令。
