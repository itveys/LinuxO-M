# Linux 面板与工具安装脚本

这是一个功能强大的Linux服务器管理脚本，可以一键安装常用的面板和工具。

## 功能特点

1. **宝塔面板安装** - 支持CentOS、Ubuntu、Debian系统
2. **哪吒监控面板** - 服务器监控和性能分析
3. **X-UI面板安装** - 多协议代理工具管理面板
4. **GitHub DNS污染检测与修复** - 检测并修复GitHub域名DNS污染
5. **Docker镜像源DNS检测与修复** - 检测并修复Docker镜像源DNS污染
6. **Docker安装** - 包含Docker常用工具
7. **服务器信息** - 显示详细的系统信息，包含实时监控功能
8. **网络测速功能** - 测试网络连通性和下载速度
9. **时间校准功能** - 手动/自动同步服务器时间，设置时区
10. **NAS配置功能** - 挂载NFS、SMB/CIFS、FTP共享到特定目录
11. **数据库备份功能** - 支持MySQL、PostgreSQL、Oracle、Redis等数据库备份
12. **Docker镜像源切换** - 切换国内镜像源，提高下载速度
13. **防火墙端口管理** - 查看和配置iptables、firewalld、ufw防火墙
14. **Docker功能菜单** - 包含多种常用服务的一键安装

## 使用方法

### 1. 下载脚本
```bash
wget https://raw.githubusercontent.com/itveys/LinuxO-M/main/linux_panel_installer.sh
```

### 2. 赋予执行权限
```bash
chmod +x linux_panel_installer.sh
```

### 3. 以root权限运行
```bash
sudo bash linux_panel_installer.sh
```

### 4. 或者直接运行
```bash
sudo ./linux_panel_installer.sh
```

## 系统要求

- **操作系统**: CentOS 7+, Ubuntu 18.04+, Debian 9+
- **内存**: 至少1GB (推荐2GB+)
- **磁盘空间**: 至少10GB可用空间
- **网络**: 需要连接互联网以下载安装包

## 功能详细说明

### 1. 宝塔面板
- 自动识别系统类型并选择正确的安装脚本
- 安装完成后显示面板访问地址
- 提供默认密码获取方法

### 2. 哪吒监控面板
- 需要提前准备好面板地址和通信密钥
- 自动下载并运行官方安装脚本
- 提供详细的配置说明

### 3. X-UI面板
- 多协议代理工具管理面板
- 支持全新安装和更新安装
- 提供默认登录信息和安全提示
- 包含完整的服务管理命令

### 4. 服务器信息与实时监控
- 显示详细系统信息（CPU、内存、磁盘、网络）
- 实时监控功能，动态显示系统资源使用情况
- 包含CPU使用率、内存使用率、磁盘空间、网络速度等

### 5. 网络测速功能
- 基础网络连通性测试（ping、DNS解析、HTTP访问）
- 下载速度测试（国内源、国际源）
- 综合网络性能测试（全面评估网络质量）

### 6. 时间校准功能
- 查看详细时间信息（系统时间、硬件时钟、时区）
- 手动设置日期和时间
- 自动时间同步（支持systemd-timesyncd、chrony、ntp）
- 设置时区（支持常用时区和自定义时区）

### 7. NAS配置功能
- 查看当前挂载点信息
- 挂载NFS共享（支持自动添加到fstab）
- 挂载SMB/CIFS共享（支持认证和匿名访问）
- 挂载FTP共享（使用curlftpfs）
- 配置自动挂载和管理共享目录

### 8. 数据库备份功能
- MySQL/MariaDB备份（支持单个数据库和所有数据库）
- PostgreSQL备份（支持远程连接和本地备份）
- Oracle数据库备份（支持expdp和exp工具）
- Redis备份（支持RDB文件备份）
- 支持定时备份任务配置

### 9. Docker镜像源切换
- 支持阿里云、腾讯云、华为云、网易云、中科大等镜像源
- 支持Docker官方源和自定义镜像源
- 测试镜像源下载速度
- 自动重启Docker服务应用配置

### 10. 防火墙端口管理
- 支持iptables、firewalld、ufw三种防火墙
- 查看当前端口状态和防火墙规则
- 开放和关闭端口
- 允许和拒绝IP地址访问
- 保存和恢复防火墙规则

### 11. DNS污染检测与修复
- 检测GitHub域名DNS污染情况
- 从多个在线服务获取最新可用IP
- 测试IP地址连通性和延迟
- 自动更新hosts文件并刷新DNS缓存
- 备份原有hosts文件

### 12. Docker安装
- 支持CentOS/Ubuntu/Debian系统
- 自动配置Docker仓库
- 安装后测试Docker运行状态
- 包含丰富的Docker管理功能

### 13. Docker功能菜单
- **ELK安装**: 配置完整的日志分析环境
- **MySQL安装**: 支持自定义密码和数据目录
- **Nginx安装**: 包含配置文件管理和网站目录
- **Redis安装**: 支持密码保护和数据持久化
- **WordPress安装**: 包含MySQL数据库的完整WordPress环境

### 14. Docker管理功能
- 查看所有容器状态
- 启动/停止/重启/删除容器
- 查看容器日志
- 进入容器终端
- 监控Docker资源使用情况

## 注意事项

1. **备份重要数据**: 在安装前请备份重要数据
2. **网络连接**: 安装过程需要稳定的网络连接
3. **系统资源**: 某些服务（如ELK）需要较多系统资源
4. **安全设置**: 安装完成后请修改默认密码
5. **防火墙**: 确保相关端口已开放
6. **权限要求**: 脚本需要root权限运行
7. **系统兼容性**: 部分功能可能因系统版本而异

## 常见问题

### Q1: 脚本无法运行怎么办？
A: 确保以root权限运行，并检查系统是否满足要求。

### Q2: 安装过程中断怎么办？
A: 重新运行脚本，大部分安装过程会自动继续。

### Q3: Docker容器无法启动怎么办？
A: 检查端口是否被占用，磁盘空间是否充足。

### Q4: 如何卸载已安装的服务？
A: 使用Docker命令或相应的卸载脚本。

### Q5: 如何查看服务的运行状态？
A: 使用 `docker ps` 查看容器状态，或使用系统服务管理命令。

### Q6: 实时监控功能如何使用？
A: 在服务器信息菜单中选择"查看实时监控"，按Ctrl+C退出。

### Q7: 如何配置定时备份？
A: 在数据库备份功能中选择"配置定时备份任务"。

### Q8: 如何切换Docker镜像源？
A: 在主菜单中选择"Docker镜像源切换"功能。
