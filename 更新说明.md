## 使用方法

### 在Linux服务器上安装X-UI:
```bash
# 1. 下载脚本
wget https://raw.githubusercontent.com/itveys/LinuxO-M/main/linux_panel_installer.sh

# 2. 赋予执行权限
chmod +x linux_panel_installer.sh

# 3. 运行脚本
sudo ./linux_panel_installer.sh

# 4. 选择第3个选项"安装 X-UI 面板"
```

### 安装过程:
1. 系统检查和安全提示
2. 依赖包安装
3. 下载官方安装脚本
4. 选择安装类型(全新/更新)
5. 执行安装
6. 显示安装结果和访问信息

## 安装后信息

### 访问信息
- 面板地址: `http://你的服务器IP:54321`
- 用户名: `admin`
- 密码: `admin`

### 常用命令
```bash
# 启动服务
systemctl start x-ui

# 停止服务
systemctl stop x-ui

# 重启服务
systemctl restart x-ui

# 查看状态
systemctl status x-ui

# 查看日志
journalctl -u x-ui -f
```

## 技术细节

### 支持的Linux发行版
- CentOS 7+
- RHEL 7+
- Fedora
- Ubuntu 18.04+
- Debian 9+

### 系统要求
- 内存: 至少512MB
- 磁盘空间: 至少2GB
- 网络: 需要连接GitHub

### 安装文件位置
- 主程序: `/usr/local/x-ui/x-ui`
- 配置文件: `/usr/local/x-ui/config.json`
- 日志文件: `/var/log/x-ui/`
- 服务文件: `/etc/systemd/system/x-ui.service`

## 故障排除

### 常见问题
1. **网络连接失败**: 检查服务器是否能访问GitHub
2. **权限问题**: 确保以root用户运行脚本
3. **端口冲突**: 检查54321端口是否被占用
4. **依赖安装失败**: 检查系统软件源配置

### 获取帮助
- 查看安装日志: `/var/log/x-ui/install.log`
- 查看服务日志: `journalctl -u x-ui`
- 访问X-UI官方GitHub: https://github.com/vaxilu/x-ui
